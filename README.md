# Русский перевод макроса lolzzn для The Forge!
* Макрос сделан для входа в локацию Volcanic Rock (Макрос может работать на любой локаций но лучше использывать его для указанной.

* Сделан lolzzn-ом, переведён dviz-ом. Макрос не выдавать за свой.
* Всегда указывайте создателя макроса если делайте по нему видео (lolzzn, не dviz-а).
* Не скачивайте макрос с других сайтов, там может быть вирус (ЭТО НЕ ОФИЦИАЛЬНЫЙ ФОРК! [ВОТ ОФИЦИАЛЬНЫЙ КОД/МАКРОС](https://github.com/lolzzn/The-Forge-Macro))
# Функций
* Поддержка Discord Веб-Хука
* AФК??
* Продаёт все "Favorite" вещи
* Продаёт легендарные предметы (пока только eye ore, uranium, lightite, mythril) до порога указанного в настройках
* Сетап гайд для одарённых
# Что нужно для работы макроса?
* AHK V2 (AutoHotKey V2)
* [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki)
* Sell Anywhere Геймпасс (лол)
# Что нужно знать до использования макроса?
* Нажмите на звёздочку по предметам/рудам которые не хотите продавать.
* UI(Интерфейса) пока нет, чтобы понять что макрос работает сверху по середине будет написано "Макрос Включен"
* [Tesseract OCR](https://github.com/UB-Mannheim/tesseract/wiki) - Библиотека для нахождения текста на экране.
